# Faculdade - BCC (UNIFAL-MG)

Repositório pessoal para organização de materiais, códigos, trabalhos e projetos do curso de **Bacharelado em Ciência da Computação**.

## Estrutura

Cada pasta representa uma disciplina, identificada pelo código oficial:

- `dce121` — Matemática Discreta
- `dce530` — Orientação a Objetos
- `dce672` — AEDs III
- `dce673` — AEDs III (Prática)
- `dce676` — Ética, Computador e Sociedade
- `dce797` — AEDs III (Teórica)

## Organização interna (padrão)
Cada disciplina pode conter:
- `aulas/` — anotações e slides
- `listas/` — listas de exercícios
- `trabalhos/` — trabalhos avaliativos
- `projetos/` — projetos maiores
- `docs/` — materiais extras

## Observações
Este repositório é de uso acadêmico e pessoal.
