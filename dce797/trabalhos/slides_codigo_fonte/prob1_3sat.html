Problema 1: 3-SAT

O 3-SAT foi o primeiro problema a ser provado como NP-Completo. Ele serve como base para quase todas as outras provas de completude.

Definição: Dada uma fórmula booleana em Forma Normal Conjuntiva (CNF), onde cada cláusula tem exatamente 3 literais, existe uma atribuição que a torna verdadeira?

Φ = (x₁ ∨ ¬x₂ ∨ x₃) ∧ (¬x₁ ∨ x₂ ∨ x₄)

• Literal: Uma variável (x) ou sua negação (¬x).
• Cláusula: Uma disjunção (OU) de literais.
• Fórmula: Uma conjunção (E) de cláusulas.

// O Teorema de Cook-Levin provou que o SAT é NP-Completo.
