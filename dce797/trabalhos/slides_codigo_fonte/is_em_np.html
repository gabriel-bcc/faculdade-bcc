Passo 1: Independent Set ∈ NP

Para provar que um problema é NP-Completo, primeiro mostramos que ele é verificável em tempo polinomial.

// Certificado: Um subconjunto de vértices S ⊆ V
function verify(G, k, S):
  if size(S) < k: return False
  for each u in S:
    for each v in S:
      if u != v and has_edge(G, u, v):
        return False
  return True
Complexidade de Verificação: O(|V|²) → Polinomial

Como podemos verificar uma solução candidata eficientemente, o problema pertence à classe NP.
