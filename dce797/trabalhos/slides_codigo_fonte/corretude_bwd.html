Corretude: Do Grafo para a Lógica

Se G possui um Independent Set S de tamanho m:

// 1. Distribuição nos Gadgets
Como existem m triângulos e |S| = m, S deve conter exatamente um vértice de cada triângulo.
// 2. Atribuição de Valores
Para cada vértice v ∈ S, atribuímos o valor Verdadeiro ao literal correspondente.
// 3. Consistência Lógica
Devido às arestas de consistência, S nunca conterá x e ¬x simultaneamente. A atribuição é válida.
// 4. Satisfatibilidade
Como cada triângulo (cláusula) tem um representante em S, cada cláusula terá pelo menos um literal verdadeiro.

Conclusão: A fórmula Φ é satisfatível.
