Problema 2: Independent Set

Definição Formal

Dado um grafo não direcionado G = (V, E) e um inteiro k, existe um subconjunto de vértices S ⊆ V tal que:

1. O tamanho do conjunto é pelo menos k: |S| ≥ k.

2. Nenhum par de vértices em S é adjacente: ∀ u, v ∈ S, (u, v) ∉ E.

Entrada: Grafo G, Inteiro k

Saída: Sim/Não (Problema de Decisão)

Conceito Visual:

Vértices que não compartilham arestas.
O objetivo é encontrar o maior conjunto possível.

// Este é o problema que provaremos ser NP-Completo via redução do 3-SAT.
