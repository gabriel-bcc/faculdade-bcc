Passo 2: A Redução Polinomial

Para provar que o Independent Set é NP-Completo, transformamos uma instância de 3-SAT em uma de Independent Set em tempo polinomial.

Instância 3-SAT
Fórmula Φ com m cláusulas
f(Φ) ➔
Instância Indep. Set
Grafo G e k = m

• Cláusulas: Mapeadas para subgrafos específicos (Gadgets).

• Literais: Mapeados para vértices dentro dos gadgets.

• Consistência: Arestas adicionais impedem escolhas contraditórias.

• Objetivo: Φ é satisfatível ⇔ G tem conjunto independente de tamanho m.
