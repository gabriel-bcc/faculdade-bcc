Corretude: Da Lógica para o Grafo

Se Φ é satisfatível ⇒ G tem Independent Set de tamanho m

1. Se Φ é satisfatível, existe uma atribuição que torna cada cláusula Verdadeira.

2. Em cada cláusula Cᵢ, pelo menos um literal deve ser verdadeiro. Escolhemos um desses literais por cláusula.

3. No grafo G, selecionamos os m vértices correspondentes a esses literais escolhidos.

4. Por que é um Independent Set?

• Não há arestas de triângulo entre eles (escolhemos apenas um por triângulo).

• Não há arestas de consistência (não podemos escolher x e ¬x, pois ambos seriam verdadeiros).
