Arestas de Consistência Lógica

Apenas criar triângulos não é suficiente. Precisamos garantir que a escolha de literais seja logicamente consistente.

A Regra:
Adicionamos uma aresta entre cada par de vértices que representam literais contraditórios (ex: x e ¬x).

Por que isso é necessário?
Em um conjunto independente, não podemos escolher dois vértices conectados. Isso impede que o algoritmo selecione x=V e x=F simultaneamente.

Essas arestas conectam vértices de triângulos diferentes, "amarrando" a lógica de toda a fórmula.

Restrição de Variável:

x
¬x

// Impede a escolha de ambos no Independent Set

// Esta etapa garante que a solução do grafo seja uma atribuição válida para a fórmula.
